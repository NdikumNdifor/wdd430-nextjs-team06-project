# Handcrafted Haven

Handcrafted Haven is a full-stack web application designed to connect artisans and crafters with customers who value unique handmade products. The platform serves as an online marketplace where creators can showcase their craftsmanship, manage product listings, and engage with a growing community of buyers.

This project is being developed as a collaborative BYU-Idaho group project focused on modern full-stack web development practices, teamwork, and cloud deployment.

## Features

- Seller profiles for artisans and crafters
- Product listings with images, descriptions, and pricing
- Product browsing and category filtering
- Product reviews and ratings
- Responsive and accessible user interface
- Secure authentication and user management
- Cloud deployment with Vercel
